## Build Setup Frontend

``` bash
# install dependencies
npm install

# build
npm run build

# serve with live-server at localhost:9876
npm run start
```

> You can edit the config to change the endpoints

``` bash
/front/config/application.js
```

## Build Setup Websockets

``` bash
 # go to directory
 cd /websockets
    
 # install dependencies
 npm install
 
 # start web server
 npm run start
 ```
